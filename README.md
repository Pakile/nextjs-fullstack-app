# Next.js full-stack application example 

A full-stack web application made with Next.js and lowdb.

## Requirements
```
Node ^16.19.1
yarn
```

## Installation
1\. Clone project

On your console :
```
git clone git@github.com:eliseekn/nextjs-fullstack-app.git project-name
```

2\. Install packages dependencies

On your console :
```
cd ./project-name
yarn && yarn dev
```

## Tests

On your console :
```
yarn test
```
